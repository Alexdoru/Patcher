<p>Patcher is a Forge mod created for the intent of fixing Vanilla bugs, Forge bugs, adding Quality of Life enhancements, and improving performance.
<br>
<br>
<b>Patcher replaces the following mods.</b>
<ul>
  <li><a href="https://sk1er.club/mods/case_commands">Case Commands</a></li>
  <li><a href="https://sk1er.club/mods/command_patcher">Command Patcher</a></li>
  <li><a href="https://sk1er.club/mods/compactchat">Compact Chat</a></li>
  <li><a href="https://sk1er.club/mods/cross_chat">CrossChat</a></li>
  <li><a href="https://2pi.pw/mods/fastchat">FastChat</a></li>
  <li><a href="https://sk1er.club/mods/item_optimization">Item Optimizations</a></li>
  <li><a href="https://prplz.io/memoryfix/">Memory Fix</a></li>
  <li><a href="https://sk1er.club/mods/mousebindfix">Mouse Bind Fix</a></li>
  <li><a href="https://prplz.io/mousedelayfix/">Mouse Delay Fix</a></li>
  <li><a href="https://hypixel.net/threads/forge-modification-noclosemychat-for-mc-1-8.1260752/">NoCloseMyChat</a></li>
  <li><a href="https://sk1er.club/mods/resourceexploitfix">Resource Exploit Fix</a></li>
  <li><a href="https://www.curseforge.com/minecraft/mc-mods/vanilla-enhancements">Vanilla Enhancements</a></li>
  <li><a href="https://skyerzz.com/minecraft/mods/voidchat/">VoidChat</a></li>
</ul>
<br>
<b>Patcher contains the following features.</b>
<ul>
  <li>Sound Customization - Change the volume of any sound available, from 0 to 2x the normal volume. Accessible by /patchersounds</li>
  <li>Fullscreen Fix - Allow the screen to be resized after toggling fullscreen.</li>
  <li>Item Searching - Minecraft doesn't stop searching for other items to combine despite being a full stack. </li>
  <li>Reset Death Timers - Allow for respawning when toggling fullscreen on the "You Died!" menu.</li>
  <li>Command Handling - Fix Forge's command handler not checking for a '/' at the start of a command.</li>
  <li>Case Insensitive Commands - Allow for case insensitivity.</li>
  <li>Inventory Position - Stop potion effects from shifting your inventroy to the right.</li>
  <li>MouseDelayFix - Stop your actual position from being a tick ahead of your crosshair.</li>
  <li>Arm Position - Reset the player state properly once mounting an entity, fixing weird arm rotation.</li>
  <li>Head Rotations - Properly rotate the users head while mounting an entity.</li>
  <li>Fullbright - Remove lighting updates, increasing visibility.</li>
  <li>Sky Height - Set the sky height to 0, removing void flickering.</li>
  <li>Instant World Swapping - Remove waiting times between swapping worlds.</li>
  <li>Nausea Effect - Remove the nether portal appearing when clearing nausea.</li>
  <li>Scoreboard Patch - Fix scoreboard spamming logs with errors.</li>
  <li>Disable Achievements - Remove achievement notifications.</li>
  <li>Container Backgrounds - Remove the dark background inside a container.</li>
  <li>Fire Overlay Height - Change the height of the ingame fire overlay.</li>
  <li>Remove Water Overlay - Remove the water texture overlay when underwater.</li>
  <li>Chat History Length - Change how many messages you can scroll back past.</li>
  <li>FOV Modifier - Allow for modifying FOV change states.</li>
  <li>Sprinting FOV - Change your FOV when sprinting.</li>
  <li>Bow FOV - Change your FOV when pulling back a bow.</li>
  <li>Speed FOV - Change your FOV when having the speed effect.</li>
  <li>Slowness FOV - Change your FOV when having the slowness effect.</li>
  <li>Tab Opacity - Change tab list opacity, based on 0 to 100%.</li>
  <li>Pack Images - Change all pack icons to 64x64 to improve memory usage.</li>
  <li>Toggle Tab - Hold tab open with a single keypress.</li>
  <li>Crosshair Perspective - Remove the crosshair when in third person.</li>
  <li>Remove Tall Grass - Stop tall grass/double tall plants from rendering.</li>
  <li>Transparent Chat - Remove the background from chat.</li>
  <li>Tab Height - Move the tab overlay down 16 pixels when there's an active bossbar.</li>
  <li>Disable Armorstands - Stop armorstands from rendering.</li>
  <li>Disable Item Frames - Stop item frames from rendering.</li>
  <li>Disable Grounded Arrows - Stop arrows that are in the ground from rendering.</li>
  <li>Disable Attached Arrows - Stop arrows that are attached to a player from rendering.</li>
  <li>Disable Moving Arrows - Stop arrows that are airborne from rendering.</li>
  <li>Disable Skulls - Stop skulls from rendering.</li>
  <li>Disable End Portals - Stop end portals from rendering.</li>
  <li>Disable Shadowed Text - Remove shadows from text.</li>
  <li>Show Own Nametag - See your own nametag in third person.</li>
  <li>MouseBindFix - Fixes an issue where keybinds bound to mouse buttons do not work in inventories.</li>
  <li>OptiFine Zoom Adjustment - Scroll when using OptiFine's zoom to adjust the zoom level.</li>
  <li>OptiFine Zoom Sensitivity - Remove the smoothing from OptiFine's zoom.</li>
  <li>Compact Chat - Clean up chat by stacking duplicate messages.</li>
  <li>Number Ping - Show a readable number in tab instead of bars.</li>
  <li>Numerical Enchantments - Use normal numbers instead of roman numerals on enchants.</li>
  <li>Startup Notification - Notify how long the game took to startup with a notification.</li>
  <li>Keep Shaders on Perspective Change - Keep the shaders you're currently using while also being able to toggle perspective.</li>
  <li>Better Keybind Handling - Make keys re-register when closing a GUI, like in 1.12+.</li>
  <li>Damage Glance - View the damage value of the currently held item above your hotbar.</li>
  <li>Item Count Glance - View the amount of the currently held item above your hotbar.</li>
  <li>Enchantment Glance - View the enchantments of the currently held item above your hotbar.</li>
  <li>Protection Percentage - View how much total protection you have inside the inventory menu.</li>
  <li>Projectile Protection Percentage - View how much total projectile protection you have inside the inventory menu.</li>
  <li>Chat Position - Move the chat up 12 pixels to stop it from overlapping the health bar, as done in 1.12+.</li>
  <li>Chat Timestamps - Add timestamps before a message.</li>
  <li>Transparent Nametags - Remove boxes around nametags.</li>
  <li>Arrow Lighting - Stop attached arrows from lighting up other entities.</li>
  <li>Entity Culling - Stop entities that aren't visible to the player from rendering.</li>
  <li>Drop Entire Stack Keybind - Allow for dropping an entire stack for MacOS users, needs to be held down.</li>
</ul>
<br>
<b>Patcher also contains the following features behind the scenes.</b>
<ul>
  <li>Forge Optimization - <a href="https://github.com/MinecraftForge/MinecraftForge/commit/5a48ca99b6787c7f811045d1f98b26db6ce073b7#diff-a27ff85fc320f1d8269b133701c1c5b2">Faster fluid checking.</a></li>
  <li>Forge Optimization - <a href="https://github.com/MinecraftForge/MinecraftForge/pull/3907">Optimized character stripping.</a></li>
  <li>Forge Optimization - <a href="https://github.com/MinecraftForge/MinecraftForge/pull/5706">Persistent Chunk Check Performance.</a></li>
  <li>Forge Optimization - <a href="https://github.com/MinecraftForge/MinecraftForge/pull/3267">Fix typo in sky rendering causing tons of BlockPos allocations.</a></li>
  <li>Forge Optimization - <a href="https://github.com/MinecraftForge/MinecraftForge/pull/5766">Fix unclosed stream in GuiModList.</a></li>
  <li>Forge Optimization - <a href="https://github.com/MinecraftForge/MinecraftForge/pull/5766">Fix unclosed stream in AnvilChunkLoader.</a></li>
  <li>Forge Optimization - <a href="https://github.com/MinecraftForge/MinecraftForge/commit/3a48a9cd731238c2a5f664362fb073732b426ef5#diff-9687ac6b081c5d4b560e95a42620a355">Skip searching for mods in the JAVA_HOME directory.</a></li>
  <li>Forge Optimization - <a href="https://github.com/MinecraftForge/MinecraftForge/pull/4938">Clear out stored errors after model loading finishes.</a></li>
  <li>1.9 Skin Support - Allow for transparency on secondary skin layers, as done in 1.9+</li>
  <li>Resource Optimization - Cache resources to reduce loading startup & pack loading times.</li>
  <li>Head Optimization - Cache custom head layers, improving performance.</li>
  <li>Settings Enhancement - Don't change mipmap levels until user leaves GUI, stopping accidental mipmap level changes & freezing the game as a result.</li>
  <li>Instant Language Swapping - Swap languages instantly, removing freezing from accidentally changing your language.</li>
  <li>Instant Dimension Swapping - Remove the loading screen from switching worlds/dimensions, making dimension swapping instant.</li>
  <li>Fix Unclosed Server Stream - Release server icons buffer, fixing a memory leak.</li>
</ul>
